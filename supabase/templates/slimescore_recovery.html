Nieuw wachtwoord kiezen 🔐

Gebruik de code hieronder om op SlimeScore.com een nieuw wachtwoord te kiezen. Open SlimeScore, kies “Wachtwoord vergeten?” → “Ik heb de resetmail al ontvangen”, vul je e-mailadres en deze code in en kies je nieuwe wachtwoord.

Code uit deze resetmail

Kopieerbare code: Tik of druk lang op deze code om ’m te kopiëren:

{{ .Token }}

Dit is een tijdelijke code uit je resetmail. Dit is niet de vaste inlogcode die een kind van de beheerder krijgt.

Zie je de mail niet? Check ook Spam of Ongewenst. Lukt het wijzigen niet? Neem contact op met de poulebeheerder.

Heb jij dit niet aangevraagd? Dan kun je deze mail negeren. Je bestaande wachtwoord blijft geldig.
